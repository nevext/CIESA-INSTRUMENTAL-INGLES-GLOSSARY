# CIESA Instrumental English Glossary

This repository contains an academic project developed for the **Instrumental English** course at CIESA.  
The goal of this project is to create a glossary with useful and interesting terms to support students in their studies.

## About the Project
- **Purpose:** Provide a simple glossary of technical and academic terms in English.  
- **Audience:** Students of CIESA, especially those enrolled in the Instrumental English discipline.  
- **Format:** Web-based glossary built with HTML, CSS, and JavaScript.  

## Features
- Search bar to look up terms.  
- Organized definitions for quick reference.  
- Expandable structure to add new terms freely.  
- Responsive design (optimized for desktop use).  

## Technologies Used
- **HTML5** for structure  
- **CSS3** for styling  
- **JavaScript** for interactivity
